# BakalarskaPracaDashborad

Aplikácia je implementovaná v jazyku Python s využitím knižnice Streamlit. Pred spustením je potrebné mať nainštalovanú verziu Python 3.8 alebo novšiu a nainštalovať všetky závislosti príkazom  `pip install streamlit pandas plotly requests` .
Aplikácia sa spúšťa príkazom `streamlit run Dashboard.py` .
